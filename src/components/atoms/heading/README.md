# About `Heading` component

## It takes two props

1. `type`: This props take various headings type such as h1 to h6 for <h1> to <h6> headings.
2. `children`: Take children props which display as children of heading tags.

## How to render `Button` component

- Firstly, import the component
- Secondly, use that component and pass props such as: [<Heading type="h1">Next</Heading>]
- `type` props takes `h1` to `h6`
